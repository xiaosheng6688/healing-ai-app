GitHub Pages 部署指南

1注册 GitHub 账号
如果你还没有 GitHub 账号：
1. 打开 github.com/signup
2. 输入邮箱、设置密码、选择用户名
3. 完成邮箱验证

如果已有账号，直接跳到第2步
2创建新仓库
1. 登录 GitHub 后，点击右上角 + 号
2. 选择 New repository
3. 填写信息：
  • Repository name: healing-ai-app
  • Description: AI情绪陪伴应用（可选）
  • 选择 Public
  • 勾选 Add a README file
4. 点击 Create repository
3上传文件
文件位置：
C:\Users\Administrator\.qclaw\workspace\healing-ai-app\docs\index.html

操作步骤：
1. 在新仓库页面，点击 Add file
2. 选择 Upload files
3. 把上面的 index.html 文件拖进去
4. 点击 Commit changes
4启用 GitHub Pages
1. 点击仓库顶部的 Settings 标签
2. 左侧菜单点击 Pages
3. Source 部分选择 Deploy from a branch
4. Branch 选择 main
5. 文件夹选择 / (root)
6. 点击 Save

等待 1-2 分钟，页面会显示你的链接
🎉

部署完成！

你的链接将是：
https://你的用户名.github.io/healing-ai-app/

开始创建仓库 →

• 部署完成后，任何设备都能访问你的应用
• 修改文件后重新上传，网站会自动更新
• 完全免费，无需服务器
• 支持自定义域名（可选）
